Habilidad Correlation Weaver para OpenClaw.

## Problema que resuelve
Encuentra correlaciones estadísticas y patrones no obvios.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque de seguridad primero
- Soporte de rollback

## Uso
/correlation-weaver

## Ejemplo
```
Analizar código para problemas de análisis.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
